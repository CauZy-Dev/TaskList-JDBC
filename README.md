Conceito:

Sistema gerenciador de lista de tarefas armazenado em um banco de dados.

Tecnologias:

Java com api JDBC

MySql

Aplicação:

Usuário poderá:

-Adicionar tarefa 

-Modificar tarefa 

-Remover tarefa 

-Mostrar a lista de tarefas

-Mostrar tarefa específica
